## Sprint 유효성 검사

이벤트가 발생할 때 작동하는 함수를 핸들러라 부른다.

btn이 눌렸을 때 작동하게 만드는 3가지 방법

```js
btn.onclick = function() {
  console.log('작동할 동작을 작성')
}
```

```js
btn.addEventListener('click', function() {
  console.log('작동할 동작을 작성')
})
```

```js
function handler() {
  console.log('작동할 동작을 작성')
}
btn.onclick = handler;
```
